# Sticky Monster Lab Clone Web

#### 1. 프로젝트명

    - Sticky Monster Lab Clone Web
    
#### 2. 프로젝트 소개

    - Sticky Monster Lab 웹사이트를 클론하여 만든 웹사이트 입니다.

#### 3. 구현 플랫폼 

    1) OS
        - windows, macOS, Linux에서 모두 동작 

    2) Front-end 기술 및 라이브러리  
        - HTML5, CSS3, JavaScript ES6 표준으로 구현 

    3) Code Editor
        - Visual Studio Code 1.41
